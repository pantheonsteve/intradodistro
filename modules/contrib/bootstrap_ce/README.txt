-- Overview --

This module allows you set a Bootstrap Carousel as a formatter of 'entity_references' field and 'image' fields.
That way you can display entities in the desired view mode or images in the desired image style.
This module comes with a simple Bootstrap carousel template that you can easily override.

-- Requirements --

A bootstrap theme. Available at: https://www.drupal.org/project/bootstrap 

-- Installing --

Enable the module.
You should now have available a Bootstrap Carousel field formatter.

-- Overriding output --

You can override the twig template file (bootstrap-ce-formatter.html.twig) located in the module's 
templates directory.
